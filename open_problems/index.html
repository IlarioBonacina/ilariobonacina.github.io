Open Problems

This is an (unordered) list of open problems from proof complexity that I find interesting. This list of problems might overlap with Alexander Rasborov’s list of open problems in proof complexity. Please let me know if you find errors or something listed as an open problem but it was solved. The last time this page was updated was on Feb 11, 2021.

On the Pigeonhole Principle

  • Are minimal size resolution refutations of \(\mathsf{PHP}^{n+1}_n\) regular?

  • Let \(S_n\) be the minimal size of a resolution refutation of \(\mathsf{PHP}^{m}_n\). What is the value of \(\limsup_{n\rightarrow \infty}\frac{\log_2\log_2 S_n}{\log_2 n}\)? We know it is between \(\frac{1}{3}\) and \(\frac{1}{2}\).

  • Consider \(\mathsf{ofPHP}^{cn}_n\) for some constant \(c\), i.e. the onto-functional version of PHP, and its refutations in \(k\)-DNF resolution. Is it true that \(\mathsf{ofPHP}^{cn}_n\) needs size \(\mathrm{e}^{n/(\log n)^{\mathcal{O}(k)}}\) to be refuted in \(k\)-DNF resolution? We know this lower bound for \(\mathsf{PHP}^{cn}_n\).

  • Prove that \(\mathsf{PHP}^{n^2}_n\) require super-polynomial \(2\)-DNF resolution refutations.

  • Prove that for every \(m\), \(\mathsf{PHP}^{m}_n\) requires polynomial calculus + resolution refutations of size \(2^{\Omega(n)}\)

  • Prove that \(\mathsf{PHP}^{2n}_n\) require super-polynomial depth-\(d\) frege refutations.

On \(k\)-clique

  • Let \(G\) be an Edros-Renyi random graph sampled from \(\mathcal{G}(n,1/2)\). W.h.p. \(G\) does not have a \(\lceil 2\log n\rceil\)-clique. Consider a propositional formula \(\mathsf{clique}(G,k)\) encoding the statement “\(G\) contains a \(k\)-clique”. Is it true that resolution refutations of \(\mathsf{clique}(G,\lceil 2\log n\rceil)\) must have size \(n^{\Omega(\log n)}\)?

  • Let \(G\) be a graph. Is it true that if \(G\) is \(k\)-clique free and with no independent set of size \(k\), then resolution refutations of \(\mathsf{clique}(G,k)\) must have size \(n^{\Omega(k)}\)?

On \(k\)-coloring

  • Let \(G\) be an an Edros-Renyi random graph sampled from \(\mathcal{G}(n,1/2)\). W.h.p. the coloring number of \(G\) is \(\frac{n}{2\log(n)}\). Let \(k<\frac{n}{2\log(n)}\) and consider the natural propositional encoding of the statement “\(G\) is \(k\)-colorable”. Let \(\mathsf{Col}(G,k)\) be this formula. Is it hard to refute in polynomial calculus? For instance for \(k=\frac{n}{4\log(n)}\).

  • What about the multivalued-encoding of the statement “\(G\) is \(k\) colorable” using the \(k\)-th roots of unity? I.e. the value of each variable encodes directly the color. Let this set of polynomials be \(\mathsf{Col}^\prime(G,k)\). Is it hard for polynomial calculus?

On random formulas and random \(3\)-XOR

  • Are random \(3\)-CNF formulas hard for Cutting Planes? We know this is true for random \(\Theta(\log n)\)-CNFs with appropriate clause density.

  • Random \(3\)-CNF formulas with \(n\) variables and \(\mathcal{O}(n^{1.5-\epsilon})\) many clauses must have exponential size resolution refutations. On the other hand random \(3\)-CNF formulas with \(n\) variables and \(\Omega(n^{2}/\log n)\) clauses have polynomial size resolution refutation. Close the gap, somehow.

  • Let \(\varphi\) be a random \(3\)-XOR instance s.t. the constraints-variables incidence graph is a \((\Omega(n),c)\)-boundary expander. Is it hard to refute \(\varphi\) in polynomial calculus (over a field of characteristic \(p\neq 2\)) when \(c<1\) is a small constant?

On Tseitin Formulas

  • We know Cutting Planes has quasi-polynomial refutations of Tseitin formulas. Is this tight?

  • We know Tseitin formulas on a constant degree random graph with \(n\) vertices require monomial space \(\Omega(\sqrt{n})\) to be refuted in polynomial calculus + resolution. Is this tight?

On SETH-like lower bounds

  • Is there any unsatisfiable \(k\)-CNF formula \(\phi\) in \(n\) variables such that all resolution refutations of \(\phi\) must have size \(2^{n(1-c_k)}\) where \(c_k\) is a function of \(k\) going to \(0\) as \(k\) goes to \(\infty\)? This is known to be true for regular resolution and slightly stronger sub-systems of resolution.

  • In addition to the open question above, can we prove \(c_k= \frac{c}{k}\) for some constant \(c\)?

  • Is there any unsatisfiable \(k\)-CNF formula \(\varphi\) in \(n\) variables such that all SOS refutations of \(\varphi\) have degree at least \(n(1-c_k)\) where \(c_k\) is a function of \(k\) going to \(0\) as \(k\) goes to \(\infty\)?

On ordering principles

  • Prove that all the polynomial calculus + resolution refutation of the dense linear ordering principle, \(\mathsf{DLO}_n\), must have size \(n^{\omega(1)}\). This problem is interesting in the measure that it is a quite simple formula that eludes all the known techniques to prove degree (and size) lower bounds in polynomial calculus + resolution.

On structural results

  • Does the monotone feasible interpolation hold for the Lovasz-Schrijver proof system?

  • Are there any non-trivial lower for space in Cutting planes with bounded coefficients? If we allow unbounded coefficients then every unsatisfiable formula has a refutation of constant space (missing reference).

  • Is there any relationship between the degree and the monomial space in polynomial calculus? That is something similar to the width-clause space inequality that holds for resolution.

  • Prove any super-linear (in the number of variables) total space lower bound in polynomial calculus + resolution. E.g. for random \(k\)-CNFs on \(n\) variables, is it true that they require total space \(\Omega(n^2)\) to be refuted in polynomial calculus + resolution?

  • Let \(\varphi\) be an unsatisfiable CNF formula, let \(M\) be minimal monomial space of a polynomial calculus + resolution refutation of \(\varphi\) and \(W\) be the minimal width of a resolution refutation of \(\varphi\). Is it true that \(M\geq \Omega(W)\)? So far we know \(M \gtrsim \Omega(\sqrt{W})\) (missing reference).

  • Is there a propositional formula encoding some natural combinatorial principle exponentially separating resolution from regular resolution? We have examples of the separation but via formulas composed with some gadget, so, well, not so natural.

Big “classical” open problems in proof complexity

  • Prove super-polynomial size lower bounds for \(\mathsf{AC}_0[p]\)-Frege.

  • Prove super-polynomial size lower bounds for \(\mathsf{TC}_0\)-Frege.

  • Prove super-polynomial size lower bounds for Frege.
